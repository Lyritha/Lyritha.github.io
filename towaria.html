Towaria

Tower-defence micro-game

Gameplay & Mechanics

Towaria is a tower defence game that focuses on punishing gameplay. You need to avoid letting enemies get to the end of the path and killing you.

As the player you can place towers that auto-kill enemies, you have to kill all enemies to progress to the next wave, and reaching the final wave of this level progresses you to the next level.

My contribution

My job within the project was mostly working on the core mechanics of the game like allowing the player to place the towers or the statistics.

Outside of my main task i also worked on 3D models and UI polish. I created the ground tiles, a tower and some enemies.

When it comes to the ui polish i mostly just passed through the UI once the functionality was done to make it fit the rest of the UI properly.

Challenges of the project

The main challenges of the project in my experience was keeping the game optimized, and sometimes communication between teammates.

When it came to optimization, the challenges i faced did give me a boost in my skills to optimize code (through using the unity Jobs system) for tasks that involve a great amount of AIs doing the same movement.

I did struggle with teamwork, our group did have a scrum master, but there was a lack of a teamleader when it comes to project management.

About the code

This piece of code is a unity job, jobs allow you to do multi-threading without having to thread manage yourself.

This specific job moves all the enemies in our game across the path towards the goal, it also rotates the enemy when needed (if it goes around a corner).
